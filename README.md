# week1-mechanical-design
set up three CAD tools — Tinkercad, Autodesk Fusion, and FreeCAD — and completed a beginner 3D modeling exercise in each, including designing a nameplate, a bracket, and a cylindrical shaft. The technologies used include Tinkercad for browser-based modeling, Autodesk Fusion for parametric CAD, and FreeCAD for open-source offline design.
